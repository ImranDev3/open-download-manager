## ✨ Features

- **⚡ Lightning Fast:** Accelerated downloading technology.
- **🎨 Modern UI:** Sleek, dark-themed UI built with React and Framer Motion.
- **📁 Smart Categories:** Automatically organizes downloads into Documents, Music, Videos, and Programs.
- **⏸️ Resume Capability:** Pause and resume downloads without losing data.
- **💻 Cross-Platform:** Available on Windows, macOS, and Linux natively.

## 🛠️ Built With

- **Electron.js** (Desktop Architecture)
- **React 19 & Vite** (Ultra-fast UI rendering)
- **Vanilla CSS** (Custom premium dark theme)

## 🚀 How to Run Locally (For Developers)

1. **Clone the repository:**
   ```bash
   git clone https://github.com/ImranDev3/open-download-manager.git
   cd open-download-manager
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Start the Desktop App:**
   ```bash
   npm run electron:start
   ```
